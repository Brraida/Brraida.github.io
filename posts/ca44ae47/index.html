InstallGitBySourceCode

环境

腾讯云、centos7

卸载

​ 可能你为了某种需求（比如必须更好地管理文件）而要卸载git，重新安装git。

一、

​ 若通过yum安装git，那么只需要执行

yum remove git #卸载git
which git
#或者
whereis git #查询git残留

二、

​ 若通过源码安装git

which git
#或者
whereis git #查询git位置
#去相应位置删除git相关文件
rm -rf git

​ 最好可以备份一下，那么就可以大胆一下去删除了。

安装

​ 回归正题。

下载

wget https://www.kernel.org/pub/software/scm/git/git-2.9.5.tar.gz

P.S. 可以在这里查找最新版本的git下载，然后替换一下git的版本数字就行。

安装依赖

yum -y install curl-devel expat-devel gettext-devel openssl-devel zlib-devel gcc perl-ExtUtils-MakeMaker

​ 有些依赖安装git时候不会提示，比如没有curl-devel、expat-devel将会发生git:’remote-https’ is not a git command。

解压git安装包

tar -zxvf git-2.9.5.tar.gz -C /root #将文件解压到root文件夹下

配置git

cd /root/git-2.9.5
.configure -prefix=/usr/local/git/ #将资源文件集中到一个位置，便于管理

编译、安装

make && make install

配置环境变量

vim /etc/profile.d/git.sh	#不在profile中配置，在这里配置是为了便于管理（不同用户有不同的路径需求，在这里配置能够更加清晰明了）

#下方为git.sh文件内容
GIT_HOME=usr/local/git #这样做是为了使用echo查看单环境变量，大可将下方GIT_HOME取代
export PATH=$PATH:GIT_HOME/bin
export PATH=$PATH:/usr/local/git/libexe/git-core
#### wq!

source /etc/profile #环境变量生效

echo $GIT_HOME #查看环境变量

查看版本

git --verison
centos install
