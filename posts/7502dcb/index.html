rpiCamera

    使用树莓派的CSI针脚进行对树莓派摄像头的控制。

背景

  • 树莓派是3b

  • 用的操作系统是raspbian os（bulleye）

需求

    使用python对树莓派摄像头进行控制，比如拍照摄影等等操作。

步骤

    因为[官方操作系统公告](Bullseye camera system - Raspberry Pi)（bulleye版本）中说，更新了树莓派摄像头的部分，因此使用不了picamera库，即不能利用python对摄像头进行一些操作。所以以“曲线救国”方式，利用OpenCV库进行摄像头的操作。（有点牛刀小试的感觉，不过能很好的满足我小小的需求）

配置

  1. 将摄像头连接好树莓派。

  2. 在树莓派终端输入sudo raspi-config，选择interfaces，打开Legacy camera

  3. 检查在/dev/下是否存在video0

  4. 输入允许树莓派进行使用树莓派摄像头模块指令：

    modprobe bcm2835-v4l2

  5. 将树莓派模块bcm2835-v4l2加入/etc/modules中

    sudo vim /etc/modules

# 在该文件下填入即可，该目的是配置开机自动加载
bcm2835-v4l2

  6. 重启

以上我就可以找到video0了。因为开启了Legacy camera1，将不能使用树莓派官方指令来进行操作摄像头了，因为没有那个需求，也没有进行下一步的深究。

以上的操作也可以摘下sd卡，在sd中操作（我没有操作过）

修改/boot/config.txt

start_x=1             # essential
gpu_mem=128           # at least, or maybe more if you wish
disable_camera_led=1  # optional, if you don't want the led to glow

安装opencv2

 sudo apt install python3-opencv -y

截图

import cv2 as cv
import time
import os
from datetime import date

'''
    this function complte:
    - take one picture

'''

def TakeSendPic():
    # get camera device
    cap = cv.VideoCapture('/dev/video0', cv.CAP_V4L)
    # get camera fps
    fps = cap.get(cv.CAP_PROP_FPS)

    # set image resolution
    cap.set(cv.CAP_PROP_FRAME_WIDTH, 800)
    cap.set(cv.CAP_PROP_FRAME_HEIGHT, 400)

    # initialize video argument
    # argument: save file location , edcode , fps , image resolution
    videoLocation = '/home/pi/Videos/video.avi'
    video = cv.VideoWriter(videoLocation, cv.VideoWriter_fourcc('I', '4', '2', '0'), fps, (800, 400))

    # take picture
    currentTime = int(round(time.time()*1000))
    dateTime = time.strftime('%Y%m%d%H%M%S',time.localtime(currentTime/1000))
    ret, frame = cap.read()
    if ret:
        imgName = '{}'.format(dateTime)
        imgPath = '/home/pi/Pictures/image-' + imgName + '.jpg'
        cv.imwrite(imgPath, frame) # make every frame to be picture
        video.write(frame) # video recording

    # release camera
    cap.release()

    # delete camera
    if os.path.exists(videoLocation):
        os.remove(videoLocation)
    else:
        print("The file does not exit")
