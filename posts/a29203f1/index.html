导入包中模块

isinstance() && type()

用法：

isinstance(object,classinfo)

用法：

type(object)

isinstance() 与 type() 区别：

  • type() 不会认为子类是一种父类类型，不考虑继承关系。

  • isinstance() 会认为子类是一种父类类型，考虑继承关系。

如果要判断两个类型是否相同推荐使用 isinstance()。

异常(exception)

​ 处理原则:出现异常不要紧，但要把异常影响范围限制到最小。

​ 实现要点：

  1. 正确的提示异常信息。
  2. 合理监控和设置监控范围和异常处理的措施。
  3. 使用finally从句回收系统资源。

常用的异常处理类（被动）

异常处理类名 触发场景
IndexError 索引出错触发
ValueError 类型参数正确，但返回值不符合预期是会触发
TypeError 函数参数类型不正确时会触发
ZeroDivisionError 除零出错
AttributeError 引用属性错误触发
OSError 无法完成操作系统级的任务时，会触发该类异常。(无法打开文件)
FloatingPointError 浮点类计算错误
OverflowError 数值运算时超过此种类数值的最大范围
UnicodeTranslateErrror Unicode转换时出错

Finally

Python 垃圾回收机制，只能帮我们回收变量、类对象占用的内存，而无法自动完成类似关闭文件、数据库连接等这些的工作。

​ 作用：资源回收(比如关闭文件、数据库连接等这些的工作)

​ 功能：无论try块是否发生异常，最终都要进入finally语句，并执行其中的代码块。

参考链接：Python try except finally：资源回收 (biancheng.net)

​ 有点意思的地方

def funcWithFinally():
    try:
        print("in try")
        return "try end"
    finally:
        print("in finally")
        return "finally end"


print(funcWithFinally())


输出结果:
in try
in finally
finally end

​ 说明第七行比第四行提前执行了。

复制和移动文件

import os, shutil


def moveFile(src, dest):
    if not os.path.isfile(src):
        print('file not exist!' + src)
    else:
        fpath = os.path.split(dest)[0]
        if not os.path.exists(fpath):
            os.makedirs(fpath)
        shutil.move(src, dest)
        print('finished moving')


def copyFile(src, dest):
    if not os.path.isfile(src):
        print('file not exist!' + src)
    else:
        fpath = os.path.split(dest)[0]
        if not os.path.exists(fpath):
            os.makedirs(fpath)
        shutil.copyfile(src, dest)
    print('finished copy')

urlopen方法

urllib.request.urlopen(url,data=None,[timeout]) #POST

​ 直接用urllib.request模块的urlopen（）获取页面，page的数据格式为bytes类型，需要decode（）解码，转换成str类型。反过来也就是说，在使用post这种方式的时候，data就要以字节形式做参传入。

import urllib.request

stockCode = '600895'
url = 'http://quotes.money.163.com/service/chddata.html'
param = bytes(urllib.parse.urlencode({'code': '0' + stockCode,
                                      'start': '20210602',
                                      'end': '20210602',
                                      'fields': 'TCLOSE;HIGH;LOW;TOPEN;CHG;PCHG;TURNOVER;VOTURNOVER;VATURNOVER'}),
              encoding='utf-8')
response = urllib.request.urlopen(url, data=param, timeout=10)
print(response.read().decode('gbk'))
response.close()

import urllib.request


def getAndSaveStock(stockCodeList, path):
    for stockCode in stockCodeList:
        url = 'http://quotes.money.163.com/service/chddata.html'
        param = bytes(urllib.parse.urlencode({'code': '0' + stockCode,
                                              'start': '20210602',
                                              'end': '202106012',
                                              'fields': 'TCLOSE;HIGH;LOW;TOPEN;CHG;PCHG;TURNOVER;VOTURNOVER;VATURNOVER'})
                      , encoding='utf8')
        urllib.request.urlretrieve(url, path + stockCode + '.csv', data=param)


stockCodeList = []
stockCodeList.append('600895')
stockCodeList.append('600007')
getAndSaveStock(stockCodeList, 'D:\\3\\')
