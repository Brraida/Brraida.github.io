使用zipfile模块

​ 用zipfile不能解压不是用传统加密的文件。经过科学上网查询得到这样的解答与解决。

​ AES：密码学中的高级加密标准（Advanced Encryption Standard，AES），又称Rijndael加密法，是美国联邦政府采用的一种区块加密标准。

​ 现在加密一般都是按这个标准。

使用pyzipper模块

​ 所以使用pyzipper模块代替zipfile模块。

​ 给上可执行正确的代码：

import pyzipper

f1 = open('D:\\1\\pwd.txt', 'r')
with pyzipper.AESZipFile('D:\\1\\test.zip', 'r') as f:
    for i in f1:
        i = i.rstrip('\n')
        f.pwd = str.encode(i)
        try:
            f.extractall('D:\\1\\2')
            print("The correct password is:" + i)
            break
        except Exception:
            pass
f1.close()

rarfile模块

​ 使用rarfile模块解压rar文件问题就多了。它解压其实借助的是电脑的解压软件的unrar.exe进行解压。(可以在pypi中查看此模块，可以得到此模块的相关信息与工作原理)

​ 然后我嫌下载unrar太麻烦，还要配置环境变量等等操作，并且我本地用的是WinRAR，所以将WinRAR所在的路径加入环境就解决了。

import rarfile 

path1 = 'D:\\1\\2.rar' 
despath = 'D:\\1\\2'  

rf = rarfile.RarFile(path1)
rf.extractall(path=despath)
rf.close()
