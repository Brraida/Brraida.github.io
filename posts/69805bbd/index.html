py-lambda

​ 今天复习到了定积分，重新学了定积分的概念。迫不及待的使用python的lambda来实验实验。

lambda[arg1[,arg2,arg3…argn]]:expression

返回的是一个方法

def getResult(a, b, n):
    fx = lambda x: x**2
    gap = (b - a) / n
    res = 0
    for i in range(n):
        res += fx(a + i * gap + gap / 2) * gap
    return res


print(getResult(1, 2, 1000))
